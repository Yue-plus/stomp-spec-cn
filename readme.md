概述
--------

![logo](./images/project-logo-vector.png)

STOMP 是一个简单（流式）面向文本的消息传输协议。

STOMP 提供了一种可互操作的连接格式， 这样 STOMP 客户端就可以与任何 STOMP 消息代理通信， 从而在多种语言、平台与代理之间提供简单而广泛的消息互操作性。
